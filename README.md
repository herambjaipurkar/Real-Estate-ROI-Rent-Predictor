Goal: Build a web app that predicts the rent or sale price of an apartment in Dubai based on community, area, and number of bedrooms.

Business Value: This tool provides a "Fair Value" estimate for investors in Dubai's high-velocity market, where prices shift weekly. Tech Stack: Python, Streamlit, Scikit-Learn, Pandas.

Data Source : Utilizing 2024-2025 Transactional Data from the Dubai Land Department via Open Data Portals.
